Quer usar a internet para saber a situação das águas brasileiras? Então é a hora boa de conhecer o novo site sobre a Conjuntura dos Recursos Hídricos no Brasil – 2017!

A Agência Nacional de Águas (ANA) publicou hoje um vídeo explicativo desta ação que serve para marcar os 20 anos da Lei das Águas.

 

 

Como afirma o próprio site “Essa nona edição também traz um balanço das recentes crises hídricas e, a partir das lições aprendidas, aponta os desafios para o aprimoramento dessa importante política pública.”

Você tem duas formas digitais para acessar a publicação, uma é o PDF que pode ser baixado ou navegando no próprio site da Conjuntura http://conjuntura.ana.gov.br
